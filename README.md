
interfaz tui desarrollada en python para gestionar el diario de prácticas
sin necesidad de usar el navegador manualmente.



## Funcionalidades

Permite:

- iniciar sesión
- listar díasdel diario
- ver y editar actividades
- confirmar días cuando estén listos



## Instalación


```bash
pip install playwright textual rich
python3 -m playwright install chromium
python3 main.py
```
